# Project Assets

把项目页、数据集页、demo 截图、视频封面等素材放在这里。

如果以后你想为某篇论文做独立项目页，可以创建：

```text
projects/
  paper-name/
    index.html
```

目前建议优先准备这些素材：

- `impact/`: 从 IMPACT project page 或论文中选 1-3 张数据集示例/任务总览图。
- `mica/`: 可参考 `Kratos-Wen/MICA` 里的 `assets/mica_teaser.jpg`。
- `gear8/`: 可参考 `Kratos-Wen/Gear8` 里的 `assets/Teaser.jpg`。
- `rohoi/`: 准备一张 benchmark overview 或 robustness examples。
- `microg4m/`: 准备一张 microgravity action/scene teaser。
